GLOBAL DESIGN RULES:
━━━━━━━━━━━━━━━━━━━━
- Every item card always shows 
  Auto-Insurance badge and QR icon
- Empty category = Post Request form, 
  never "No results found"
- Mr. Rentano appears as small avatar 
  guide throughout all screens
- Mobile first, 390px width
- Border radius: 12px on all cards
- Green buttons only for primary CTAs
- Amber only for highlights and accents
- No dark backgrounds except 
  bottom nav active state
- Generous padding, clean white space
- Real photos in mockups, not gray boxes
- Platform feels alive even with few items —
  show request feed as main content

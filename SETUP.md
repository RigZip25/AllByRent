# AllByRent — простая инструкция (не для программистов)

Всё **уже настроено в коде**. Вам нужно только **запустить** приложение на компьютере и открыть его на телефоне. Ниже — по шагам.

---

## Что уже сделано автоматически (вам не трогать)

- Экраны онбординга, адрес, штаты США, визард из 7 шагов  
- Файл описания проекта: `PROJECT_CONTEXT.md`  
- Скрипт запуска: `dev.cmd`  
- Иконки для будущего PWA в папке `public`

---

## Шаг 1. Откройте папку проекта

Путь на вашем компьютере:

`C:\Users\Ed\Downloads\AllByRent Mobile App UI`

---

## Шаг 2. Запустите сервер (самый простой способ)

1. В проводнике Windows откройте эту папку.  
2. **Дважды щёлкните** по файлу **`dev.cmd`**.  
3. Появится чёрное окно (терминал) — **не закрывайте его**, пока смотрите приложение.  
4. Подождите 10–30 секунд, пока не появится текст вроде:

   `Local:   http://localhost:5173/`  
   `Network: http://192.168.x.x:5173/`

5. Запомните строку **Network** — это адрес для телефона.

**Если окно сразу закрылось** — см. «План Б» в конце.

---

## Шаг 3. Откройте на компьютере

1. Откройте браузер (Chrome или Edge).  
2. В адресной строке введите: **`http://localhost:5173`**  
3. Должен открыться AllByRent.

---

## Шаг 4. Откройте на телефоне (та же Wi‑Fi)

1. Телефон и компьютер в **одной Wi‑Fi**.  
2. В Safari на iPhone введите адрес из **Network** (пример):  
   **`http://192.168.1.161:5173`**  
   Цифры у вас будут свои — копируйте из окна `dev.cmd`.  
3. Используйте **`http://`**, не `https://`.  
4. Если Safari не открывает — попробуйте Chrome на Android или тот же адрес на ПК.

**Геолокация на телефоне** по `http://` часто не работает — это нормально: выберите **United States**, штат **Arkansas**, введите адрес с городом, например:  
`123 Main St, Little Rock, AR`  
и **нажмите строку в списке подсказок**.

---

## Шаг 5. Ключи для ИИ (по желанию, не обязательно сейчас)

Нужны только если хотите анализ фото и чат Rentano с Claude.

1. В папке проекта найдите файл **`.env.example`**.  
2. Скопируйте его и переименуйте копию в **`.env.local`**.  
3. Внутри вставьте свои ключи (если есть) после знака `=`.  
4. Сохраните, перезапустите `dev.cmd`.

Без ключей приложение всё равно открывается — часть ИИ-функций просто не заработает.

---

## Шаг 6. PWA «как приложение» (уже установлено в проекте)

PWA уже подключён в коде. Если вы **впервые** открываете проект на новом компьютере и dev.cmd ругается на пакеты — один раз запустите **`install-pwa.cmd`** (двойной клик), затем снова **`dev.cmd`**.

На iPhone: в Safari → «Поделиться» → «На экран Домой» (после деплоя на HTTPS; локально через `http://` — только просмотр в браузере).

## План Б — если `dev.cmd` не работает

1. Нажмите клавишу **Win**, введите **cmd**, откройте «Командная строка» или «Terminal».  
2. Вставьте и нажмите Enter (одна строка):

```text
cd /d "C:\Users\Ed\Downloads\AllByRent Mobile App UI" && npm install && npm run dev
```

3. Дождитесь `Network: http://...` — как в шаге 4.

**Если пишет «npm не найден»** — установите Node.js с https://nodejs.org (кнопка LTS), перезагрузите компьютер, повторите.

## Установка PWA («как приложение» на телефон)

В приложении уже есть **зелёная подсказка** после онбординга и на главной ленте:

- **iPhone (Safari):** Поделиться → **На экран «Домой»**
- **Android (Chrome):** кнопка **Install app** или меню ⋮ → **Установить приложение**

Подсказка **не показывается**, если человек уже открыл установленную иконку с домашнего экрана. Крестик скрывает её на 14 дней.

**Важно для пользователей:** PWA с телефона работает только по **HTTPS** (сайт на Vercel/Netlify и т.п.). Адрес `http://192.168...` с Wi‑Fi — только просмотр в браузере, без установки.

## Коротко

| Действие | Как часто |
|----------|-----------|
| Двойной клик `dev.cmd` | Каждый раз, когда хотите посмотреть приложение |
| `http://localhost:5173` на ПК | После запуска dev.cmd |
| `http://192.168.x.x:5173` на телефоне | То же, свои цифры из Network |
| `.env.local` с ключами | Когда появятся ключи API |
| `install-pwa.cmd` (если ошибка пакетов) | Один раз на новом ПК |

В Cursor я **не могу нажать кнопки на вашем ПК и телефоне** — только правлю файлы и даю команды. Запуск — только у вас, но это буквально «открыть dev.cmd и адрес в браузере».
